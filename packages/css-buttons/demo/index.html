Buttons

Use the .c-btn component for consistent button styling.

This component removes default :focus styling. You must use JavaScript to restore equivalent styling at least for keyboard-induced focus by applying the .is-focused class. Otherwise, keyboard usability and accessibility will be compromised. You can do this by using the React button component or by writing your own JavaScript similar to this.

Groups

The .l-btn-group layout class can be used as a container for grouped buttons. Use .is-selected to indicate item selection.

Use the group to combine a button with an icon button to support a split button as shown. The icon button should gain .is-active styling while the menu is open. Icon rotation is achieved by adding .is-rotated to the .c-btn__icon element.
