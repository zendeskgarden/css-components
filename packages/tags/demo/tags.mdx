# Tags

Component classes for styling your tags. Use pill or round class modifiers to
shape a tag. Round tags are intended to contain a minimal number of
characters. For example, you may want to use a round tag to show numbers 1-9
in a notification badge and then programmatically switch to a pill treatment
for displaying 10+ notifications. The tag may also contain a child
`.c-tag__avatar` element.
