# Buttons

Component classes for styling your buttons. Buttons are styled to work
together with the
[focus-visible](https://www.npmjs.com/package/focus-visible) polyfill.

## Icons

The `.c-btn--icon` modifier is used to support icon buttons. Style the child
image using the `.c-btn__icon class`. Icon rotation is achieved by adding
`.is-rotated` to the `.c-btn__icon` element.
