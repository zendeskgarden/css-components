Ranges

Use the .c-range component to style form range fields. Stateful .is-* classes are for demo purposes only and are not considered reliable for rendering checkbox behavior. However, the .is-focused class is intended to be programmatically toggled on keyboard (not mouse) focus as it is here.

Since WebKit browsers have no supporting pseudo-element for the lower range, the .c-range__input child contains corresponding background CSS whose size should be manipulated via JavaScript whenever the range "thumb" is moved. See the supporting page JS for details.
