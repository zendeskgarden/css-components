Checkboxes, Toggles, & Radios

Checkboxes showing styling for the various states. Stateful .is-* classes are for demo purposes only and are not considered reliable for rendering checkbox behavior. However, the .is-focused class is intended to be programmatically toggled on keyboard (not mouse) focus as it is here. Each state variation is shown with with corresponding .c-chk--toggle and .c-chk--radio styling.
