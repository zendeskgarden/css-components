Icons

The .c-btn--icon modifier is used to support icon buttons. Style the child image using the .c-btn__icon class. The following examples demonstrate the icon sized as default, medium, and large. (See the corresponding Garden React <IconButton> component.)

Groups

The .l-btn-group layout class can be used as a container for grouped buttons. Use .is-selected to indicate item selection. (See the corresponding Garden React <ButtonGroup> component.)

Use the group to combine a button with an icon button to support a split button as shown. The icon button should gain .is-active styling while the menu is open. Icon rotation is achieved by adding .is-rotated to the .c-btn__icon element. (See the corresponding Garden React <SplitButton> component.)
