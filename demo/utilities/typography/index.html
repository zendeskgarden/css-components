Utilities: Typography

Garden's bedrock resets apply a system font to page HTML. If not using bedrock, you can achieve the same result by using the .u-font-family-system utility (as this page does) or by leveraging Garden CSS variables to set this same system font stack in your code. Utility font sizes are based on a 1.125 modular scale. Base line height is 20px and a 20 unit vertical rhythm is suggested, but not enforced.
