Bedrock / Grid

Grid Layout

Grid layout is achieved via a top-level .l-grid class. Each column within the grid is a .l-grid__item along with a width utility class. Bedrock CSS surfaces the following width units.

There are several options available for producing fractional grid column widths. Many of these involve collapsing the space between column elements via use of negative margins. There are situations where this approach is unreliable or causes unanticipated side effects. The Bedrock grid assumes that column elements are side-by-side, eliminating whitespace.
